# Video

Container that displays a video

## Features

## Example

```
  {% include "@theme/patterns/components/video.html.twig" %}
```

## Twig parameters:

| Key           | Type   | Description                                         | Mandatory |
| ------------- | ------ | --------------------------------------------------- | --------- | --- |
| title         | string | Title of the video                                  | yes       |     |
| body          | string | Body for the video                                  | yes       |     |
| video_type    | string | Defines the type of video (iframe or hosted)        | yes       |     |
| video_src     | string | Path to the video (youtube embed or local)          | yes       |     |

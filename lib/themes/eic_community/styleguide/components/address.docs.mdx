# Address

Displays an address element.

## Example:

```
{% include "@theme/patterns/components/address.html.twig" %}
```

## Twig parameters:

| Key                          | Type   | Description                                            | Mandatory |
| ---------------------------- | ------ | ------------------------------------------------------ | --------- | --- |
| contributor.content          | string | The content within the address element.                | no        |     |
| contributor.extra_classes    | string | Defines optional classnames for the container element. | no        |     |
| contributor.extra_attributes | object | Defines optional attributes for the container element. | no        |     |

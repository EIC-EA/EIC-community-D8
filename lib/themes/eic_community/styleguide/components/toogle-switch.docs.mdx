# Toogle switch

Displays a toogle switch

## Example:

```
{% include "@theme/patterns/components/toogle-switch.html.twig" with {
  name: String,
  state: Boolean
} %}
```

## Twig parameters:

| Key            | Type   | Description                                              | Mandatory |
| -------------- | ------ | -------------------------------------------------------- | --------- |
| name          | string | name of the input                | yes       |
| state | boolean | state of the switch | yes       |

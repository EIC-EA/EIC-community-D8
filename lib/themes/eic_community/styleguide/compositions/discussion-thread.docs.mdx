# Discussion Thread

Displays the Discussion thread that could be used within the Teaser Overview composition.

## Example:

```
{% include "@theme/patterns/compositions/discussion-thread.html.twig" %}
```

## Twig parameters:

| Key               | Type     | Description                                                                                                                    | Mandatory |
| ----------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------ | --------- | --- |
| contributor_label | string   | Adjusts the semantic label of the discussion contributor.                                                                      | no        |     |
| started_by_label  | string   | Adjusts the visible label of the discussion author.                                                                            | no        |     |
| title_element     | string   | Displays the discussion title within the selected element.                                                                     | no        |     |
| from_contributor  | boolean  | Displays the styling if the discussion is from a contributor.                                                                  | no        |     |
| author            | boolean  | The author of the discussion. Should match the author component parameters                                                     | no        |     |
| tags              | object[] | Displays the defined tags within the [ECL Tag component](https://www.npmjs.com/package/@ecl-twig/ec-component-tag) parameters. | no        |     |
| members           | object   | The members that are active within the discussion. Should follow the author collection composition parameter structure.        | no        |     |
| path              | string   | The URL to the actual discussion.                                                                                              | no        |     |
| title             | string   | The title of the discussion.                                                                                                   | no        |     |
| description       | string   | The description of the discussion.                                                                                             | no        |     |
| icon_file_path    | string   | The URL for the SVG icon sprite.                                                                                               | no        |     |
| timestamp         | object   | The given timestamp of the discussion, should follow the custom timestamp component structure.                                 | no        |     |

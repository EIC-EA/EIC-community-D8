# Editorial Header

Displays a 2 column section with a title and optional meta fields within the content column.
The Editorial Header can also display optional buttons within the second column.

The displayed video is based on the existing [ECL media container](https://ecl-twig-php.netlify.app/ec/?path=/story/components-media-container)

## Example:

```
{% include "@theme/patterns/compositions/editorial-header.html.twig" %}
```

## Parameters

| Key                   | Type     | Description                                                                                                                                                   | Mandatory |     |
| --------------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------- | --- |
| title                 | string   | The title to display within the content column.                                                                                                               | no        |     |
| title_element         | string   | Use an alternative title HTMLElement.                                                                                                                         | no        |     |
| meta                  | string   | Displays the defined meta items above the title. Should contain the [Item list component](/?path=/docs/components-item-list--base) parameters.                | no        |     |
| tags                  | object[] | Displays optional tags next above the title. (Based on the [ECL tag component](https://ecl-twig-php.netlify.app/ec/?path=/story/components-tag)).             | no        |     |
| actions               | object[] | Displays optional actions next to the title. (Based on the [ECL button component](https://ecl-twig-php.netlify.app/ec/?path=/story/components-button--text)). | no        |     |
| actions[]             | object   | Should contain the properties of the ECL button component](https://ecl-twig-php.netlify.app/ec/?path=/story/components-button--text).                         | no        |     |
| parent_link           | object   | Displays a link for the parent page. Should contain the properties of the ECL link component](https://www.npmjs.com/package/@ecl-twig/ec-component-link).     | no        |     |
| description           | string   | Displays the description below the title.                                                                                                                     | no        |     |
| image                 | object   | Displays a image before the titel.                                                                                                                            | no        |     |
| stats                 | object[] | The optional clickable statistics.                                                                                                                            | no        |
| stats[].label         | string   | The semantic label for the current statistic.                                                                                                                 | no        |
| stats[].value         | string   | The displayed statistic value.                                                                                                                                | no        |
| stats[].icon          | object   | The displayed ECL icon for the current statistic.                                                                                                             | no        |
| stats[].updates       | object   | Displays an optional update indicator for the current statistic.                                                                                              | no        |
| stats[].updates.label | string   | The semantic label for the statisicic update indicator.                                                                                                       | no        |
| stats[].updates.value | string   | The displayed value for the statisicic update indicator.                                                                                                      | no        |

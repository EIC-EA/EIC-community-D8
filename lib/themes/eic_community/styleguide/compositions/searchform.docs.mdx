# Searchform

Displays the global searchform.

## Example:

```
{% include "@theme/compositions/searchform.html.twig" %}
```

## Parameters

| Key               | Type   | Description                                                               | Mandatory |     |
| ----------------- | ------ | ------------------------------------------------------------------------- | --------- | --- |
| tooltip           | object | Displays the custom tooltip component above the fields.                   | no        |     |
| field             | object | Defines the ECL input properties for the searchfield.                     | no        |     |
| submit            | object | Defines the properties for the submit button component.                   | no        |     |
| suggestion        | string | Displays a search suggestion in editable content.                         | no        |     |
| container element | string | Use an alternative container element instead of the default form element. | no        |     |

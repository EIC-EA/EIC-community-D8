# Global Header

The custom global header that is based on the original [ECL Site header])(https://www.npmjs.com/package/@ecl-twig/ec-component-site-header) component.

## Features

- Excluded the searchfield since it will be present within the custom mainmenu.
- Rewritten original JS functionality for the language selector & list composition.
- Option to only define a single logo if the specific `src_desktop` & `src_mobile` are not defined.
- Implements optional login button & active user with updates & dropdown options.

# Card Wrapper

Displays the custom version of the [ECL-Twig Card component](https://ecl-twig-php.netlify.app/ec/?path=/story/components-card).

## Example:

```
{% include "@theme/patterns/compositions/card-wrapper.html.twig" %}
```

## Features

- Uses the same data structure as the default [ECL Twig Card component](https://ecl-twig-php.netlify.app/ec/?path=/story/components-card--tile)
- Support to include 1 or multiple `authors` within the card, that will be displayed below the default fields.

## Parameters

| Key                      | Type     | Description                                                                                               | Mandatory |     |
| ------------------------ | -------- | --------------------------------------------------------------------------------------------------------- | --------- | --- |
| card                     | object   | Defines the structure of the card.                                                                        | yes       |     |
| card.type                | string   | Card type (tile)                                                                                          | no        |     |
| card.description         | string   | Displays the card description below the title                                                             | no        |     |
| card.meta                | string[] | Displays a basic list of meta values.                                                                     | no        |     |
| card.title               | string   | The actual title of the card                                                                              | no        |     |
| card.image               | object   | Displays the image above the title.                                                                       | no        |     |
| card.image.src           | string   | The path of the card image                                                                                | -         |     |
| card.image.alt           | string   | Defines the alt attribute for the optional image                                                          | -         |     |
| card.tags                | object[] | Displays one or multiple optional tags.                                                                   | no        |     |
| card.tags[].label        | string   | The label op the defined tag.                                                                             | -         |     |
| card.tags[].path         | string   | The URL for the defined tag.                                                                              | -         |     |
| card.infos               | object[] | Displays one or multiple info field with optional icon                                                    | no        |     |
| card.infos[].label       | string   | The label of the info field                                                                               | -         |     |
| card.infos[].icon        | object   | Displays the optional icon aside from the defined info label                                              | -         |     |
| card.infos[].icon.type   | string   | Defines the type of the icon that should match the ECL Icon spec.                                         | -         |     |
| card.infos[].icon.name   | string   | Name of the defined icon that should exist within the given icon path                                     | -         |     |
| card.infos[].icon.path   | string   | The URL of the SVG icon sprite where the defined name should exist in.                                    | -         |     |
| card.links               | object[] | Displays the defined links in a vertical flow at the bottom of the defined card                           | no        |     |
| card.links[].label       | string   | Displays the label of the defined link                                                                    | -         |     |
| card.links[].path        | string   | The URL for the defined link.                                                                             | -         |     |
| card.authors             | object[] | Displays the optional authors for the defined card.                                                       | no        |     |
| card.authors[].author    | string   | The name of the defined author                                                                            | -         |     |
| card.authors[].image     | object   | Displays the author image if defined. The initials of the given name will be used if no image is defined. | -         |     |
| card.authors[].image.src | string   | The path of the author image.                                                                             | -         |     |
| card.authors[].image.alt | string   | Defines the alt attribute for the defined author image.                                                   | -         |     |
| icon_file_path           | string   | The URL to the inline SVG sprite.                                                                         | no        |     |
| extra_classes            | string   | Optional classes for the actual ECL Twig card component.                                                  | no        |     |
| extra_attributes         | string   | Optional attributes for the actual ECL Twig card component.                                               | no        |     |

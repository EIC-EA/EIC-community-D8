# Teaser Overview

Displays an overview with content items within a rows or columns that should be used within an overview page.

## Parameters

| Key                | Type   | Description                                                                   | Mandatory |     |
| ------------------ | ------ | ----------------------------------------------------------------------------- | --------- | --- |
| extra_classes      | string | Inserts additional classnames within the root composition.                    | no        |     |
| no_items_available | string | Informative text that will be displayed when defined if no items are defined. | no        |     |
| items              | string | The items that will be displayed.                                             | no        |     |
| items[].content    | string | The renderable field.                                                         | no        |     |

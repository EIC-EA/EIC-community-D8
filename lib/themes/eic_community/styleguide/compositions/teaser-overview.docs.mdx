# Teaser Overview

Displays an overview with content items within a rows or columns that should be used within an overview page.

## Parameters

| Key                             | Type          | Description                                                                                                                    | Mandatory |     |
| ------------------------------- | ------------- | ------------------------------------------------------------------------------------------------------------------------------ | --------- | --- |
| extra_classes                   | string        | Inserts additional classnames within the root composition.                                                                     | no        |     |
| no_items_available              | string        | Informative text that will be displayed when defined if no items are defined.                                                  | no        |     |
| items                           | object[]      | The items that will be displayed.                                                                                              | no        |     |
| items[].content                 | string        | The renderable field.                                                                                                          | no        |     |
| amount_options                  | object        | Displays the amount options within a [ECL Select Twig component](https://www.npmjs.com/package/@ecl-twig/ec-component-select). | no        |     |
| amount_options.title            | string        | The title as select label for the amount option.                                                                               | no        |     |
| amount_options.total_of         | string        | The prefix label that defines the semantic amount.                                                                             | no        |     |
| amount_options.total            | number        | The amount of available entries.                                                                                               | no        |     |
| amount_options.items            | object[]      | Defines the select options for the amount option.                                                                              | no        |     |
| amount_options.items[].value    | string/number | The value attribute of the current amount option.                                                                              | no        |     |
| amount_options.items[].label    | string/number | The readable label of the current amount option.                                                                               | no        |     |
| active_filters                  | object        | Displays the selected filters as tags.                                                                                         | no        |     |
| active_filters.title            | string        | The semantic title for the active filters option.                                                                              | no        |     |
| active_filters.items            | object[]      | The active filters displayed as (ECL Twig Tag component)[https://www.npmjs.com/package/@ecl-twig/ec-component-tag].            | no        |     |
| active_filters.items[].label    | object[]      | The label of the active filter.                                                                                                | no        |     |
| active_filters.items[].value    | object[]      | The filter value of the active filter placed as data-value attribute within the remove button.                                 | no        |     |
| sort_options                    | object        | Displays the sort options for the current overview.                                                                            | no        |     |
| sort_options.title              | string        | The semantic title for the active filters option.                                                                              | no        |     |
| sort_options.items              | object[]      | The sort options displayed as (ECL Twig Link component)[https://www.npmjs.com/package/@ecl-twig/ec-component-link].            | no        |     |
| sort_options.items[].label      | string        | The sort option label.                                                                                                         | no        |     |
| sort_options.items[].path       | string        | The URL of the sort option.                                                                                                    | no        |     |
| sort_options.items[].icon       | object        | Follows the [ECL Twig icon component](https://ecl-twig-php.netlify.app/ec/?path=/story/components-icon--general) properties.   | no        |     |
| interface_options               | object        | Displays the view options for the current overview.                                                                            | no        |     |
| interface_options.title         | string        | The title for the view options list.                                                                                           | no        |     |
| interface_options.items         | object[]      | Defines each view option for the overview.                                                                                     | no        |     |
| interface_options.items[].label | string        | The label for the current view option.                                                                                         | no        |     |
| interface_options.items[].path  | string        | The URL that should display the view option.                                                                                   | no        |     |
| interface_options.items[].icon  | object        | Follows the [ECL Twig icon component](https://ecl-twig-php.netlify.app/ec/?path=/story/components-icon--general) properties.   | no        |     |

# Attachment

Displays a semantic timestamp with icon.

## Example:

```
{% include "@theme/patterns/components/timestamp.html.twig" %}
```

## Twig parameters:

| Key            | Type   | Description                                              | Mandatory |
| -------------- | ------ | -------------------------------------------------------- | --------- |
| label          | string | Displays the timestamp of the component.                 | yes       |
| icon_file_path | string | Enables the usage of the ECL icons within the component. | no        |

# Overview Header

Displays the overview header that can have an optional background image.

## Example:

```
{% include "@theme/compositions/overview-header.html.twig" %}
```

## Parameters

| Key       | Type   | Description                                 | Mandatory |     |
| --------- | ------ | ------------------------------------------- | --------- | --- |
| title     | string | The title that will be displayed.           | yes       |     |
| image     | object | Displays the optional background image.     | no        |     |
| image.src | string | The URL of the actual image.                | -         |     |
| image.alt | object | The alt attribute for the background image. | -         |     |

# Text Block

Displays a multicolumn section with editable content and a single media container.

## Example:

```
{% include "@theme/patterns/compositions/introduction-header.html.twig" %}
```

## Parameters

| Key                    | Type     | Description                                                                                                                      | Mandatory |     |
| ---------------------- | -------- | -------------------------------------------------------------------------------------------------------------------------------- | --------- | --- |
| title                  | string   | Displays a main title before the editable content                                                                                | no        |     |
| title_element          | string   | Defines an alternative title HTMLElement.                                                                                        | no        |     |
| items                  | object[] | Should contain an array with one ore multiple content fields                                                                     | no        |     |
| items[].content        | render   | Should contain the rendered editable content of the current field                                                                | no        |     |
| items[].call_to_action | object   | Displays an optional call to action button below the editable field                                                              | no        |     |
| media                  | object   | Should contain the [ECL media container](https://ecl-twig-php.netlify.app/ec/?path=/story/components-media-container) parameters | no        |     |

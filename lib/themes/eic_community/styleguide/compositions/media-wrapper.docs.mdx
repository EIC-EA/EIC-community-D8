# Media Wrappers

Displays a full width media container as a section block that is based on the existing [ECL media container](https://ecl-twig-php.netlify.app/ec/?path=/story/components-media-container).

## Example:

```
{% include "@theme/patterns/compositions/media-wrapper.html.twig" %}
```

## Parameters

| Key   | Type   | Description                                                                                                                      | Mandatory |     |
| ----- | ------ | -------------------------------------------------------------------------------------------------------------------------------- | --------- | --- |
| media | object | Should contain the [ECL media container](https://ecl-twig-php.netlify.app/ec/?path=/story/components-media-container) paramaters | no        |     |

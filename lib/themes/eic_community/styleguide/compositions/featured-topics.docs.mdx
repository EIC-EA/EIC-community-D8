# Featured Topics

Displays the defined topics in a tag overview.

## Example:

```
{% include "@theme/components/featured-topics.html.twig" %}
```

## Twig parameters:

| Key     | Type     | Description                                                                                                                               | Mandatory |
| ------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------- | --------- |
| title   | string   | Displays an optional title before the overview.                                                                                           | no        |
| items   | object[] | Should contain an array with entries that match the [ECL Twig Tag](https://ecl-twig-php.netlify.app/ec/?path=/story/components-tag--link) | yes       |
| items[] | object   | Should be an object of type [ECL Twig Tag](https://ecl-twig-php.netlify.app/ec/?path=/story/components-tag--link)                         | yes       |

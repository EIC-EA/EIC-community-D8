# Top Menu

Displays a top menu section that should display a custom description and 1 level navigation menu.

## Parameters

| Key          | Type     | Description                                                                                                                                                                      | Mandatory |
| ------------ | -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------- |
| description  | string   | Inserts an optional description after the flag.                                                                                                                                  | no        |
| items        | object[] | Displays one or multiple menu items to the right.                                                                                                                                | no        |
| items[].link | object   | The link object for the current menu item. Should follow the [ECL Twig Link component](https://www.npmjs.com/package/@ecl-twig/ec-component-language-list) component parameters. | no        |
